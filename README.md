# DoctorDocumentation
ASP.NET CORE 3.1 Web Application

## How to install

Open your terminal (CMD or Bash) and execute the next lines:
```bash
git clone https://github.com/tehica/DoctorDocumentation
```

## How to run

After running the next line, open http://localhost:5000:
```bash
dotnet run
```
